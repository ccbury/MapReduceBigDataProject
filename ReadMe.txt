The mapReduce implementations are split into tho different folders
	SaleRecords and StateNames

Within these folders are all required files to run the map reduction on that dataset.

Ensure Python 3.7 is installed on the computer running the MapReductions
Ensure the pandas library is installed on the system
	pip install pandas

SalesRecords
	Simply run the SalesMapReducerun.bat file to start the MapReduction

Statenames
	simply run the NameMapReduceRun.bat file to start the MapReduction
